Admin Module

WarrantySystem\Admin

The Admin Module provides administrative functionality for the Warranty Management System, including user management, system configuration, and access control.

Module Overview

The Admin Module is the central control panel for system administrators. It provides functionality for managing users, system settings, categories, and viewing system statistics through the dashboard.

Key Files

dashboard.php
Path: /admin/dashboard.php

Main dashboard file that displays system statistics and key performance indicators.

users.php
Path: /admin/users.php

Handles user management functionality, including creating, editing, and deleting users.

categories.php
Path: /admin/categories.php

Manages claim categories, including creating, editing, and deleting categories.

settings.php
Path: /admin/settings.php

Handles system-wide settings and configuration options.

User Management

User Roles

The system supports the following user roles:

  • Administrator: Full access to all system functions
  • Manager: Access to reports and claim management, but not system settings
  • CS Agent: Limited access to claim processing and customer service functions

Dashboard

Dashboard Components

The dashboard provides the following key information:

  • Total claims by status (new, in_progress, on_hold, approved, rejected)
  • Recent claims (last 10)
  • Claims by category (pie chart)
  • Claims by month (line chart)
  • SLA compliance rate
  • Average resolution time
